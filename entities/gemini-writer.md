# gemini-writer (Doriandarko/gemini-writer)

An AI writing agent powered by Gemini 3 Flash that creates novels and stories autonomously (⭐276). The simplest among all the novel writing agents surveyed.

## Overview
Created by Doriandarko, the same author as kimi-writer. This agent uses Google's Gemini 3 Flash model for novel generation. It's essentially a single Python script that feeds prompts to Gemini.
