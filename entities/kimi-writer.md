# kimi-writer (Doriandarko/kimi-writer)

An AI writing agent powered by kimi-k2-thinking that creates novels and stories autonomously (⭐564). A simpler Python-based single-agent approach compared to multi-agent systems like InkOS.

## Overview
Created by Doriandarko (known for several AI agent projects), kimi-writer uses Moonshot AI's kimi-k2-thinking model for deep reasoning in novel generation. It's a relatively simple agent — a single Python script that prompts an LLM to write novels with reasoning.

## Limitations
- Simpler architecture — single agent, not multi-agent pipeline
- No built-in audit/revision loop
- No style imitation engine
- No web UI or TUI
- No longer actively maintained (last update January 2026)
- Relies on kimi-k2-thinking API (less commonly available than OpenAI/Anthropic)
