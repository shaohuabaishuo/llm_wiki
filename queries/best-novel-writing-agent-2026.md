# Query: Best Open-Source Agent for Chinese Novel Writing (April 2026)

## Original Question
"我现在想要进行小说创造，帮我找到目前最适合的开源agent"
("I want to create a novel, help me find the most suitable open-source agent")

## Answer
**InkOS** (Narcooo/inkos) is the best choice by a wide margin.

## Ranking

| Rank | Agent | Stars | Why |
|------|-------|-------|-----|
| 🥇 | [[inkos-novel-agent]] | ⭐4,830 | Chinese native, one-command install, 7-agent pipeline, most features |
| 🥈 | [[ai-novel-writing-assistant]] | ⭐725 | Full web UI, beginners welcome, Chinese-first |
| 🥉 | [[autonovel]] | ⭐727 | Python pipeline, English-only, proven 79K-word output |
| 4 | [[kimi-writer]] | ⭐564 | Single-script, inactive |
| 5 | [[gemini-writer]] | ⭐276 | Minimal, abandoned at creation |

## Why InkOS Won
- **Stars:** 4,830 — 6x more than the nearest competitor
- **Activity:** Updated daily (last update: 2026-04-22)
- **Install:** `npm i -g @actalk/inkos` — ready in seconds
- **Chinese:** Full native support — CLI, docs, TUI, Studio all in Chinese
- **Pipeline:** 7 specialized agents per chapter with 33-dimension audit
- **Export:** EPUB for reading on phone/Kindle
- **Genres:** xuanhuan, xianxia, urban, sci-fi, and more
